## Description
This is a console-based quiz application written in Python.  
Users can choose a subject and difficulty level, answer multiple-choice questions, and view statistics such as past scores and highest score achieved.

## Features
- Subjects: Maths, Science  
- Difficulty levels: Easy, Medium, Hard  
- Randomized questions per quiz  
- Score tracking and statistics  
- View full question bank  

## How to Run

1. Make sure Python is installed.
2. Run quiz.py
